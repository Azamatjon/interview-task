## Description

This project was created as an interview task described in: [link](https://teletype.in/@sophie_ker/Oe94lWCfF8A)

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
